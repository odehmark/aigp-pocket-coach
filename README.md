# AIGP Pocket Coach

Mobile-first Progressive Web App for AIGP certification practice.

## Included
- 100 original scenario-based questions
- Domain and difficulty filters
- Detailed explanations and distractor feedback
- Progress and domain analytics stored locally
- Bookmarks
- Dark mode
- Offline PWA support

## Publish with GitHub Pages
1. Upload all files and folders in this package to the root of the repository.
2. In GitHub, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select **main** and **/(root)**, then Save.
5. Open the published URL in Safari.
6. Tap **Share → Add to Home Screen**.

Repository target:
`odehmark/aigp-pocket-coach`
